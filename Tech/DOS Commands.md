**DIR** – *Displays files and directories in the current directory.*
**CD or CHDIR** – *Changes the current directory.*
**MD or MKDIR** – *Creates a new directory.*
**RD or RMDIR** – *Removes an empty directory.*
**COPY** – *Copies files from one location to another.*
**DEL or ERASE** – *Deletes specified files.*
**REN or RENAME** – *Renames a file or directory.*
**TYPE** – *Displays the contents of a text file.*
**ECHO** – *Displays messages or turns echoing on/off.*
**VER** – *Displays the operating system version.*
**DATE** – *Displays or sets the system date.*
**TIME** – *Displays or sets the system time.*
**CLS** – *Clears the screen.*
**PROMPT** – *Changes the command prompt appearance.*
**COPY CON** – *Creates a new file from console input.*
**MOVE** – *Moves files to a new location.*
**FIND** – *Searches for a specific string in files.*
**CHKDSK** – *Checks a disk and displays its status report.*
**FORMAT** – *available commands or provides help for a command.*
