**git fetch**  
*Only downloads updates from the single remote linked to your active branch*

**git fetch --all** 
*Download the latest updates from GitHub without merging them* 

**git reset --hard branch-name** 
	*Reset your local branch to match the remote branch exactly*
	*(Replace 'main' with 'master' or your specific branch name)*

**git clean -df**
 *`-d`: Removes untracked directories.*
*`-f`: Forces the deletion.*
